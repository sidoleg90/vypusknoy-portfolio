# Выпускной портфолио · Олег

Одностраничный сайт-портфолио для выпускного курса «Архитектор нейросотрудников».
Показывает ~27 проектов, сделанных персональным AI-агентом за 7,5 недель (21 мая — 10 июля 2026):
4 флагмана + полный каталог, живые демо с QR-кодами.

## Открыть
```bash
open index.html            # macOS
xdg-open index.html        # Linux
# или сервер:
npm run serve              # http://localhost:8123
```
Работает и просто по `file://` — двойной клик по `index.html`.

## Что внутри
- **Флагманы:** Линдэйли AI Assistant (RAG/MCP/Docker), Парсер ЛПР (@parsing_lpr_bot),
  mr.Vors (живой калькулятор ковров), ШураФло (магазин + ЮKassa).
- **Каталог:** инженерия, система агентов, работа в Линдэйли, маркетинг, события, личное.
- **Живые демо:** QR на lindaily-guide, бот, калькулятор, разбор, shuraflo.ru.

## Правки
Контент — только в `data/projects.json`, потом `npm run build-data`.
Подробности для AI-агента и человека — в [AGENTS.md](AGENTS.md).

## Структура
```
index.html          каркас + дизайн-токены (:root) + все секции
app.js              рендер из window.PORTFOLIO + анимации + QR
data/projects.json  ИСТОЧНИК ПРАВДЫ (правь здесь)
data/projects.js    сгенерирован из json (грузит страница)
assets/             qr-*.svg, shot-*.png (превью), баннер
tools/gen_qr.py     генерация QR (segno)
tools/shots.py      скриншоты (playwright)
docs/brief.md       бриф и источники цифр
```

## Приватность
Показывать **локально**. Внутри реальные цифры Линдэйли — в открытый доступ не публиковать.
